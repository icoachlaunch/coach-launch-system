Visual Style Guide

The locked visual standard for every Coach Launch doc. Clean, bold, premium — built so the message hits hard and the brand is unmistakable.

Core Style Guide · Locked

This is the rulebook for the look of every guide — not the writing (that lives in the Voice Guide). Every doc inherits one stylesheet (coach-launch-guide.css) and drops its content into the same components you see below. Build once, stay on-brand forever.

▲ The one rule

Never restyle a guide by hand. If a guide needs a new component, we add it here first, then every guide gets it. That's what keeps 50 guides looking like one brand.

Color

Crimson is the hero — it drives energy and attention. Everything else steps back. A clean white canvas with charcoal text and generous space lets the crimson pop and the message hit hard. (Note: this is a light-theme override of the Style Guide's dark default — approved by Matthew.)

Charcoal#26272b — body text

Tint#f5f6f7 — section tint

✕ Never

No pastel or neon accents. No rainbow palettes. No gradients-for-decoration. Crimson is the only accent color — everything else is white, charcoal, and grey.

Typography

Montserrat for display — headings, titles, buttons, labels. Bold (700) and ExtraBold (800) for headlines. Lato for body. Bold, geometric, modern. No serifs. No scripts.

Headline · Montserrat 800

Subhead · Montserrat 700

Label · Montserrat 700 caps

Body copy is Lato, set at 17px with generous line height so long passages stay easy to read. This is the workhorse — every paragraph in a guide uses it.

Logo

The official Coach Launch logo sits at the top of every doc, centered above the title. On our light guides we use the dark-lettering version so it reads cleanly on white. The icon mark — a U-shape with an upward arrow (turnaround and upward trajectory) — is used alone for favicons and avatars.

Official logo — top of every doc

Icon mark — favicon / avatar

Note on the logo

Every guide loads the logo from your CDN URL, so it's always the current official file. To make guides fully self-contained (offline / archived), drop the PNG into /assets and point the <img src> there instead. Minimum 40px height. Use the dark-lettering logo on light backgrounds; the white version only on dark or crimson.

Good vs. bad examples

✓ Do this

"I help burnt-out real estate coaches fill a room and close 8–12 clients in 28 days."

✕ Not that

"I help entrepreneurs unlock their potential and scale to the next level."

Self-assessment checklist

  • Names a specific person, not "everyone".
  • Includes a concrete outcome with a number.
  • States the timeframe.
  • Reads out loud in one breath.
