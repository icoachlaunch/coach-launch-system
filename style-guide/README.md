# Style Guide — Coach Launch Core

The locked, canonical brand standard. Everything else in this repo follows these documents.

- **visual-style-guide.html** — how every Coach Launch doc *looks*: colors, typography, logo, and the full component library. Sourced from Brand Style Guide v2 (light-theme override, approved).
- **voice-guide.html** — the **Brand Voice** (storytelling): how you sound when selling and connecting — social, email, sales pages, scripts. Sourced from Brand Voice & Tone v1.
- **teaching-voice-guide.html** — the **Teaching Voice**: how you sound in guides, builder prompts, and how-to content — plain, direct, 5th-grade, zero jargon.

**Which voice when:** selling/connecting → Brand Voice. Instructing/building → Teaching Voice.

Both link the shared stylesheet at `../assets/coach-launch-guide.css`. The visual system lives in that one file — change it there and every guide updates at once.

**Locked** means: don't restyle or re-voice by hand. If something needs to change, change it here first, then everything downstream inherits it.
